# my-static-games

華文教學的純靜態 HTML 學習遊戲——學生玩、玩完即走，**無後端、無學情追蹤**。
適合：臨時演示、跨學校分享、不需要學情數據的場景。

🌐 部署：https://zeg5566.github.io/my-static-games/
