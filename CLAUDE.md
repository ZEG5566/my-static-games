# my-static-games — 純靜態學習遊戲

## 這個 repo 是什麼
華文教學的純靜態 HTML 學習遊戲。學生玩、玩完即走，**無後端、無數據追蹤**。
適合：臨時演示、跨學校分享、不需要學情數據的場景。

如果以後某個遊戲需要追蹤學生作答 → 該遊戲應該放到姐妹專案 `my-tracked-games/`（Firebase + 教師後台）。

## 工作注意事項
- 所有遊戲必須是**單檔 HTML**（含 inline CSS/JS），可離線打開、能 file:// 跑
- commit 訊息：`Add <课文名> — 互动游戏` 或 `Update <课文名> — <改动摘要>`
